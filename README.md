# MERCY SOLAR WORLD 🌞

A simple, clean, and responsive website for **MERCY SOLAR WORLD**, showcasing services, mission, projects, and a contact form powered by Formspree.

---

## 🚀 Live Site
Visit the site here: [https://lordszygote.github.io/mercyworld-site/](https://lordszygote.github.io/mercyworld-site/)

---

## 📂 Project Structure
- `index.html` → Main site with 5 sections (Home, Services, Mission, Projects, Contact)
- `thank-you.html` → Custom thank‑you page after form submission
- `styles.css` → Unified stylesheet for all pages
- `images/` → Placeholder images for projects (can be replaced with real backgrounds later)

---

## ✨ Features
- Responsive layout with clean navigation
- Contact form integrated with **Formspree**
- Custom thank‑you page redirect after form submission
- Unified theme colors (deep green, solar yellow, sky blue)

---

## 🛠️ How It Works
1. User visits the site (`index.html`)
2. Navigates through sections
3. Submits the contact form
4. Formspree processes the submission
5. Redirects to `thank-you.html`
6. User can return home via the button
